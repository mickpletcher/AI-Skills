# AI-Skills Repository

A comprehensive collection of reusable skills and prompts for multiple AI platforms including ChatGPT, Claude, and GitHub Copilot.

## Structure

- **shared/**: Common standards and guidelines across all platforms
- **chatgpt/**: ChatGPT-specific skills and configurations
- **claude/**: Claude-specific skills and templates
- **copilot/**: GitHub Copilot prompts and instructions
- **templates/**: Portable and platform-specific skill templates

## Documentation

See individual directories for platform-specific documentation and templates.
