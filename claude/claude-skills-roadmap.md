# Claude Skills Roadmap

A structured, high-impact set of skills for Anthropic Claude focused on automation, AI orchestration, infrastructure, and monetization.

---

## Core Dev + Automation Skills

### API Scaffold Generator
- Input: API idea or specification
- Output: Full scaffold including routes, authentication, and tests
- Extras: Postman collection, OpenAPI spec

### PowerShell Automation Builder
- Converts plain English into production-ready PowerShell modules
- Includes logging, retry logic, and error handling

### Web Automation Agent (Playwright-first)
- Generates automation scripts using browser automation
- Reduces API costs via scraping strategies
- Includes session reuse and anti-bot handling

### Spec-Driven Builder
- Reads requirements.md, spec.md, tasks.md
- Outputs working implementation aligned to spec
