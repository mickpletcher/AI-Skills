# Astrophotography Processing Skill Upgrades

## Candidate Improvements

- Add tool-specific quick-start recipes for PixInsight, Siril, APP, and DSS.
- Add narrowband palette examples for SHO, HOO, Foraxx-style blends, and naturalized SHO.
- Add FITS metadata audit script for exposure, filter, temperature, gain, dimensions, and missing frame groups.
- Add file organization script that creates standard processing folders without moving originals automatically.
- Add troubleshooting decision trees for flats, gradients, star halos, and walking noise.
- Add export presets for web, print, archive TIFF, and social platforms.
