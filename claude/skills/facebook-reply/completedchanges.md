# Completed Changes: facebook-reply

This tracked file records shipped changes for this skill.

## Completed Changes

- 2026-05-13: Added reply tone modes. The skill now supports appreciative, clarifying, humorous, and boundary setting replies based on the comment goal.
- 2026-05-13: Added context aware replies. The skill now accounts for the original post topic and who is being answered when that context is provided.
- 2026-05-13: Added reply depth control. The skill now infers concise or fuller replies to match the depth of the comment.
- 2026-05-13: Added difficult comment handling. The skill now handles skepticism, criticism, hostility, and off-topic comments without escalating.
- 2026-05-13: Added batch reply mode. The skill can now produce distinct replies for multiple comments on the same post.
- 2026-05-13: Added thread summarization mode. The skill can now detect repeated questions and draft one pinned clarification reply.
- 2026-05-13: Added sentiment tagging. The skill now tags supportive, neutral, skeptical, critical, off-topic, and hostile comments before replying.
