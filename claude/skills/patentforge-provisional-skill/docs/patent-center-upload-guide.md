# Patent Center Upload Guide

General manual filing support only. This document is not legal advice.

## Purpose

Use this guide when the draft package is ready for human review and the filer needs a plain language checklist for manual Patent Center preparation.

## Before Logging In

- Confirm the invention title used across the package
- Confirm inventor names and ownership details
- Confirm the draft text matches the intended disclosure
- Confirm figure files exist and are readable
- Confirm the filer understands this skill does not submit anything directly

## Draft Package To Review

- Invention disclosure
- Provisional style draft
- Figure list and any prepared drawings
- Prior art report notes
- Claim candidate notes
- Attorney review notes

## Manual Upload Preparation

- Confirm Patent Center account access
- Confirm the correct application path for the intended filing
- Prepare final upload files in the required formats
- Use clear final file names
- Review the bibliographic data carefully before submission

## Manual Submission Walkthrough

1. Sign in to Patent Center.
2. Start the appropriate new application workflow.
3. Enter applicant and bibliographic information carefully.
4. Upload each final file manually and verify the labels.
5. Review the entire submission before submitting.
6. Review any fee details presented in Patent Center.
7. Submit only after the responsible filer confirms the package is correct.
8. Save the confirmation receipt and submission details.

## After Submission

- Save a copy of the submission confirmation
- Save the final filed package
- Record the filing date and internal matter notes

## Reminder

- This guide is not legal advice.
- The user is responsible for final Patent Center submission.
