# PatentForge Provisional Draft Package

## 1. Invention Disclosure

Tool repositionable cable routing clip for workshop benches with a reusable clamp body, rotatable guide channel, and non marring tension interface.

## 2. Technical Summary

The concept uses a movable bench mounted clip assembly that secures one or more cables while allowing fast relocation without screws or adhesive.

## 3. Problem Solved

Fixed cable clips and ties do not adapt well to changing tool positions on active benches.

## 4. Existing Solutions

Adhesive clips, zip ties, and permanent brackets are common but are inconvenient to reposition and can damage surfaces.

## 5. Novel Feature Candidates

- Repositionable clamp body with repeat use retention
- Adjustable cable guide orientation
- Surface friendly gripping structure

## 6. Component Breakdown

- Clamp body
- Tension interface
- Cable guide channel
