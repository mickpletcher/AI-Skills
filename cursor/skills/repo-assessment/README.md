# repo-assessment

A Cursor skill for producing durable repo assessments, handoff docs, and next step guidance in active codebases.

## Source of Truth

Use [instructions.md](C:/Users/mick0/OneDrive/Documents/Code%20&%20Dev/GitHub/AI-Skills/cursor/skills/repo-assessment/instructions.md) as the editable source of truth for this skill.

## How To Implement In Cursor

### Option 1: Project Rules

1. Open the target repo in Cursor.
2. Open the project rules surface.
3. Copy the full contents of [instructions.md](C:/Users/mick0/OneDrive/Documents/Code%20&%20Dev/GitHub/AI-Skills/cursor/skills/repo-assessment/instructions.md).
4. Paste the content into the project rule.
5. Save it.
6. Test it against the current repo root and docs.

### Option 2: One Time Chat Context

1. Open Cursor Chat.
2. Paste the instruction content first.
3. Then ask Cursor to assess the open repo and produce a durable handoff.

## First Test

```text
Use repo-assessment on this repo and produce an actionable handoff with current state, risks, and the best next step.
```
