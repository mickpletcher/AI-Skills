# Gemini

This folder is reserved for Google Gemini specific skills and reusable assistant definitions.

## Intended Use

Use this platform area for Gemini focused skill wrappers, assistant instructions, Gems source material, and platform specific setup notes.

## Beginner Note

When Gemini skills are added, each skill folder should explain:

1. which file is the editable source
2. how to copy that content into a Gemini Gem or equivalent Gemini setup surface
3. how to test the setup with one simple prompt
