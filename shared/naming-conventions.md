# Naming Conventions

Standardized naming conventions for skills, files, and directories across all AI platforms.

## File Naming

- Use kebab-case for directory and file names
- Use descriptive, concise names
- Include platform prefix when platform-specific

## Examples

- `azure-deployment-skill.md`
- `python-testing-standard.md`
- `skill-standard.md`

## Directory Naming

- Keep directory names short and descriptive
- Use lowercase
- No spaces, use hyphens instead

## Metadata

All skills should include standardized metadata headers at the top of the file.
