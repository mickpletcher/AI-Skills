# Portability Guide

Guidelines for making skills and prompts portable across multiple AI platforms.

## Core Principles

1. **Platform Agnostic Language** - Avoid platform-specific syntax when possible
2. **Clear Metadata** - Include platform requirements and compatibility info
3. **Modular Design** - Structure content for easy adaptation
4. **Documentation** - Clearly note what needs to change per platform

## Platform Variations

### ChatGPT vs Claude vs Copilot

- Different prompt syntaxes
- Varying capability levels
- Distinct file format requirements

## Conversion Process

1. Identify core intent
2. Adapt to platform-specific format
3. Validate functionality
4. Document variations
