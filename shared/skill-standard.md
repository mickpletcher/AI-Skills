# AI Skill Standard

Guidelines and standards for creating portable AI skills across platforms.

## Overview

This document defines the baseline structure and requirements for all skills in this repository.

## Requirements

- Clear metadata headers
- Consistent naming conventions
- Platform-specific adaptations documented
- Version tracking
- Usage examples
